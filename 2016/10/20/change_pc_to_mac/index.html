将HEXO从windows迁移到Macbook

发表于 2016-10-20   |   分类于 Computer Skills   |  

将hexo的主题和source文件拷贝过来,其他内容手动配置,类此此过程参看JinyaoLi’s Blog

本文前提：

  • Mac系统安装了Node.js和Git
  • 本文中所有命令均在terminal中执行

拷贝需要的文件

需要拷贝一下文件夹：

  • hexo/source
  • hexo/themes/你的主题文件夹

和以下文件:

  • hexo/_config.yml
  • packge.json

在Macbook部署新的Hexo

  • 1运行 npm install hexo -g , npm install hexo --save。
  • 2在你想要部署hexo的文件夹运行命令: hexo init,创建一个新的Hexo目录。
  • 3将拷贝的packge.json文件复制到新的hexo目录下,覆盖原来的文件。
  • 4执行 npm install 安装依赖包。
  • 5拷贝剩下的两个文件夹和_config.yml到对应位置。
  • 6运行 hexo g 生成博客。
  • 7运行 hexo s -o 查看你的博客。
